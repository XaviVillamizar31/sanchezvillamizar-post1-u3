# Perfil Personal con CSS3

**Estudiante:** Xavi Villamizar  
**Programa:** Ingeniería de Sistemas  

## Descripción
Este proyecto consiste en una página web de perfil personal desarrollada con HTML5 y CSS3. La página incluye una sección de perfil, una sección de habilidades y un formulario de contacto, aplicando conceptos como selectores CSS avanzados, el modelo de caja (box model), posicionamiento y estilos accesibles para formularios.

## Instrucciones de ejecución
1. Clonar o descargar el repositorio.
2. Abrir la carpeta del proyecto en Visual Studio Code.
3. Ejecutar el archivo **index.html** utilizando la extensión **Live Server**.
4. El navegador mostrará la página de perfil con todas sus secciones.
